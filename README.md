# Loan Portfolio Analytics using Microsoft Fabric

## Project Overview

This project implements an end-to-end Loan Portfolio Analytics solution using Microsoft Fabric.

### Technologies
- Microsoft Fabric
- OneLake
- Lakehouse
- PySpark
- Data Pipelines
- Delta Tables
- Star Schema
- Power BI

## Architecture

Bronze → Silver → Gold → Power BI

## Current Progress

- Bronze layer ingestion completed
- Silver layer transformation completed
- Gold star schema completed
- ETL pipeline created
- Audit logging implemented
- Data profiling completed
- Data dictionary created
- Referential integrity validation completed

Power BI dashboard is under development
